# Osmosis LBP Simulator

## Requirements

- [Go](https://golang.org/doc/install)
- [Node](https://nodejs.org)

## Run locally

Clone repo:

```sh
git clone git@github.com:public-awesome/lbp-simulator.git
cd lbp-simulator
```

Start backend:

```sh
make run
```

Start web app (in another terminal):

```sh
cd app
yarn install
yarn dev
```

Open [localhost:3000](localhost:3000)
